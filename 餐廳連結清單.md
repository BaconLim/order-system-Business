# 🍽️ 餐廳網頁連結清單 (Restaurant Links)

這裡整理了目前系統中所有店家的存取連結。

## 📍 本機測試連結 (Local Testing)
如果您直接用瀏覽器開啟檔案，請在網址列的 `index.html` 後面加上以下參數：

- **王府肉燥飯** (測試用)
  - 連結: `index.html?store=wangfu`
  
- **漢堡王**
  - 連結: `index.html?store=burgerking`

- **桃樂絲手作便當**
  - 連結: `index.html?store=dorothy`

- **千葉素食便當 (總店)**
  - 連結: `index.html` (預設) 或 `index.html?store=default`

---

## 🌐 上線後的連結範例 (Deployment)
假設您將網頁上傳到 GitHub Pages (例如網址是 `https://bacon.github.io/order-system/`)，那麼連結會變成：

1. **王府肉燥飯**: `https://bacon.github.io/order-system/?store=wangfu`
2. **漢堡王**: `https://bacon.github.io/order-system/?store=burgerking`
3. **桃樂絲手作便當**: `https://bacon.github.io/order-system/?store=dorothy`
4. **總店**: `https://bacon.github.io/order-system/`

> [!TIP]
> 這些連結參數 (`?store=...`) 是對應到 `store_data.js` 裡面的設定。如果要新增店家，記得在 JS 檔與這裡同步更新。
