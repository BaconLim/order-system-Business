# 系統操作與擴充指南

本指南說明如何檢視目前系統狀況，以及如何透過我 (AI 助手) 擴充新的店家資訊。

## 1. 檢視現有店家
若要切換不同店家，請在網址後方加上 `?store=[代碼]` 参数：

- **千葉素食便當 (總店)**: `index.html` (預設)
- **王府肉燥飯**: `index.html?store=wangfu`
- **桃樂絲手作便當**: `index.html?store=dorothy`
- **漢堡王**: `index.html?store=burgerking` (新!)

## 2. 如何新增店家 (指令範本)
若您未來想要新增其他店家，請提供以下資訊給我：

1.  **店家代碼 (ID)**: 英文代號，例如 `kfc` 或 `starbucks`。
2.  **店家名稱**: 顯示在標題的中文名稱。
3.  **菜單內容**: 列出所有品項與價格。
4.  **(選填) 風格顏色**: 若有特定的品牌色 (例如紅色、藍色)。
5.  **(選填) 封面圖片**: 若有指定的圖片網址。
6.  **(選填) 營業時間**: 若需自訂外送時間 (預設 10:30-12:30)。

### 推薦的指令格式
您可以直接複製以下格式對我下達指令：

> 請幫我新增一個店家，代碼是 `[新的代碼]`，店名叫做「`[店名]`」。
>
> 菜單如下：
> - `[品項1]` `[價格]`
> - `[品項2]` `[價格]`
> ...
>
> 請幫我設定它的代表色是 `[顏色]`。
> 營業時間是 `[HH:mm]` 到 `[HH:mm]`。

---
**範例指令：**
> 請幫我新增店家，代碼 `burgerking`，店名「漢堡王」。
> 菜單有：華堡 $150、小華堡 $100、薯條 $45。
> 風格顏色用紅色。
> 營業時間改為 09:00 到 20:00。

## 3. 設定多店家推播通知 (Pushover)
若您希望不同店家的訂單通知不同人 (例如王府訂單通知王老闆)，請依照下列步驟修改後端程式碼：

1.  開啟您的 Google Apps Script 專案。
2.  找到 `STORE_MANAGERS` 設定區塊。
3.  將對應店家的 User Key 修改為該店長的 Key。

```javascript
  const STORE_MANAGERS = {
    'wangfu': '王老闆的_USER_KEY',      
    'burgerking': '店長的_USER_KEY', 
    'default': '您的_USER_KEY'
  };
```
修改完成後，請務必點選 **「部署」 -> 「新版本」** 才會生效喔！
